# online-meeting

## to start the app 

1.clone  repo 
```sh
  git clone https://github.com/saeedkhal/webRTC.git 
```
2.run server

* get into server folder
    ```sh
    cd server
    ```
* install dependancy
  ```sh
    npm install 
  ```
* install finally run server
  ```sh
    nodemon server.js 
  ```
3-run front-end 
  * got back to the root folder
    ```sh
    cd ..
    ```
  * got get into front-end folder
    ```sh
    cd frontend-side
    ```
  * install dependancy
    ```sh
      npm install 
    ```
  * install finally run server
    ```sh
      npm start 
    ```
